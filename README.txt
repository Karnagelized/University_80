=== Стек ===
1. Язык программирования - Python версии не ниже 3.11.0. Требуется установленный pip версии не ниже 23.0.0.
2. Среда для просмотра кода - PyCharm(Рекомендуется), Блокнот, Sublime Text 3, NotePad++ и другие.





=== Как запустить проект ===
Есть несколько вариантов как посмотреть проект:
    1. PyCharm
    2. Скомпелированный EXE файл
    3. Посмотреть видеоролик по проекту
Ниже будут предоставлены подробные инструкции по запуску проекта.





=== Как проверить версию Python и наличие pip ===
    Чтобы проверить корректность версий, следует открыть проект через PyCharm, далее перейти в
Terminal(Открыть терминал можно через сочетание ALT + F12) и написать следующие команды:
    1. python -V (Проверьте, что версия Python не ниже 3.11)
    2. pip list (Проверьте, что версия pip не ниже 23.0.0)

    Если у Вас отсутсвует pip или версия Python ниже 3.11, рекомендуктся запускать проекта через EXE файл,
находящийся в папке "Скомпилированный EXE файл". Либо Вы можете просмотреть видеоролик с названием "RESULT.mp4".





=== Как запустить проект через PyCharm ===
    Примечание: Для корректной работы проекта требуется Python версии 3.11.0, а также установленный pip версии
не ниже 23.0.0. (Как проверить версии смотрите выше!).
    
    Если все версии удовлетворяют условиям запуска проекта, то переходим к запуску проекта. Открываем main.py
через PyCharm, запускаем проект через комбинацию "SHIFT + F10". 





=== Как запустить проект через EXE файл ===
    В папке проекта переходим в директорию с названием "Скомпилированный EXE файл". В ней открываем "University.exe".





=== Как найти видеоролик с результатом проекта ===
    В папке проекта открываем видеоролик с названеим "RESULT.mp4".
